Preguntas Frecuentes

¿Para qué edad está diseñada esta plataforma?

Para todos los niños que estén cursando primero de primaria.

¿Necesito crear una cuenta para usar el material?

Si, es necesario crear una cuenta para tener acceso al material.

¿Qué temas se cubren?

Temas de matemáticas básicas como sumas, restas, recta numérica, longitudes, mándalas, juegos, etc.

¿Puedo usarlo desde un celular o Tablet?

No, solo se puede usar desde un ordenador o portátil.

¿Tiene algún costo el material?

No, solo se necesita crear una cuenta.
